## 📚 What I Learned

Through this project, I learned and practiced:

- Writing **semantic HTML** for better structure and accessibility
- Using **CSS padding and margin** correctly for spacing
- Centering images and content using **Flexbox**
- Building a clean card-style layout
- Organizing CSS for readability and maintainability

---

## 🚧 Challenges Faced

One challenge was managing spacing between sections and ensuring the layout looked consistent across different screen sizes.  
I overcame this by using proper padding, margin, and Flexbox alignment instead of fixed positioning.
